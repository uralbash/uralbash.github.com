Основы коллективных систем разработки ПО¶

Обзор¶

Для организации контролируемого процесса разработки, необходимо множество таких систем как: система управления версиями (VCS), система управления кодом (SMS), taskmanager, bugtracker и т.д.

Системы отслеживания ошибок и постановки задач:

  • Trac
  • Bugzilla
  • Redmine

Системы контроля версий:

  • git
  • mercurial
  • subversion

Социальные сети для разработчиков:

  • github.com
  • bitbucket.org
  • sourceforge.net

Мы будем использовать git и github - самые распространенные инструменты среди программистов на данный момент.

Git¶

http://progit.org/book/ru/ - основная документация по Git. Нас будут интересовать первые три главы: введение, основы Git, ветвления в Git(слияния).

Данный учебник является репозитарием на github и хостится как статический сайт при помощи сервиса Pages.
